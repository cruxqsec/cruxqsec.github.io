# My personal blog

This is my personal blog. It's using the amazing [Congo](https://jpanther.github.io/congo/) theme.

## Modifications

### Theme
I'm using a custom `montenegro` theme that I found. You can find it at `assets/css/schemes/montenegro.css`

### Zooming.js
I'm using this amazing Javascript zooming library called - you guessed it - [Zooming](https://kingdido999.github.io/zooming/docs/#/)
